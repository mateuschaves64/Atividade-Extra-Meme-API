API utilizada: Repositório meme API (Por D3vd)
Endpoint: https://github.com/D3vd/Meme_Api?utm_source=chatgpt.com

Descrição: A API contém memes retirados de 3 subreddits (Dankmemes, wholesomememes, me_irl) cada um contendo dimensões diferentes que fazem a páginna redimensionar o tamanho,
no projeto, foi utilizado o endpoint //meme-api.com/gimme/wholesomememes que filtra para apenas aparecer memes do subreddit wholesomememes (pois os outros contém conteúdos
sugestivos ou ofensívos).
